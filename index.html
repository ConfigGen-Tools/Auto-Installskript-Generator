Automatisierter Software-Installer für Windows (Batch)

Erstellt ein Skript zur automatischen Installation von Programmen nach der Windows-Einrichtung.

Fügen Sie Programme hinzu, die automatisch nach der Windows-Installation gestartet werden sollen.

1. Installation hinzufügen

C:\Temp\

Achtung: Setup-Typ prüfen!

  • **MSI-Dateien** sind für die Automatisierung (Silent Install) am zuverlässigsten.
  • **EXE-Setups** (insb. App Installer/MSIX) können im Batch-Modus fehlschlagen oder benötigen spezielle Parameter.

Generiert: `start /wait "" "C:\Temp\[Ihr Pfad]"`

2. Installationsschritte

Fügen Sie Schritte hinzu, um das Skript zu starten.

Das generierte Skript führt die Installationen automatisiert im Hintergrund aus.

Wie Sie das Skript verwenden

  1. Klicken Sie auf den Button **"Als .bat herunterladen"** und speichern Sie das Skript als Standardsoftware-Installer.bat.
  2. Kopieren Sie die **.bat-Datei** auf den Ziel-PC (z.B. auf den Desktop oder ein temporäres Laufwerk).
  3. Stellen Sie sicher, dass alle lokalen Setup-Dateien (falls Sie welche hinzugefügt haben) im Pfad C:\Temp\ auf dem Ziel-PC abgelegt sind. **WICHTIG:** Das Skript erstellt diesen Ordner nicht selbst.
  4. Klicken Sie mit der rechten Maustaste auf die .bat-Datei und wählen Sie **"Als Administrator ausführen"**, um die Installation zu starten.

Wichtiger Hinweis

Das Skript nutzt `start /wait`, um jede Installation abzuwarten. Es wird erst nach der letzten Installation (und dem Aufräumen) abgeschlossen. Bitte warten Sie, bis das Konsolenfenster sich schließt.
